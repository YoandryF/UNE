# Contribuir a UNE Web

¡Gracias por tu interés en contribuir! Este proyecto ayuda a los cubanos a controlar su consumo eléctrico.

## Cómo contribuir

1. **Fork** el repositorio
2. Crea un **branch** desde `main`: `git checkout -b feature/mi-mejora`
3. Realiza tus cambios
4. **Commit** con mensaje descriptivo: `git commit -m "feat: descripción"`
5. **Push** a tu fork: `git push origin feature/mi-mejora`
6. Abre un **Pull Request** hacia `main`

## Reglas

- Los PRs requieren aprobación del mantenedor antes de merge
- No se puede pushear directamente a `main`
- Mantener el código en un solo archivo HTML (single-file app)
- Usar vanilla JS (sin frameworks ni dependencias externas)
- Probar en móvil antes de enviar PR

## Convenciones de commits

- `feat:` nueva funcionalidad
- `fix:` corrección de bug
- `docs:` documentación
- `style:` estilos/UI
- `refactor:` refactorización sin cambio funcional

## Ideas bienvenidas

- Mejoras de accesibilidad
- Soporte para más tipos de tarifa (comercial, etc)
- Traducciones
- Optimizaciones de rendimiento
- Mejoras visuales

## Reporte de bugs

Abre un Issue describiendo:
- Qué esperabas que pasara
- Qué pasó realmente
- Navegador y dispositivo
- Pasos para reproducir
